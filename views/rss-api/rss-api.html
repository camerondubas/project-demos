How It Works:

Input a URL for any RSS Feed and receive a JSON Formatted version. It is CORS Enabled so this API can be used on strictly front-end projects without any need for server-side logic.

Note:

This demo only displays the first 10 items in a feed, however the API supports retrieving feeds in their entirety.

Sample Feeds:

  • http://www.feedforall.com/sample.xml
  • http://www.feedforall.com/sample-feed.xml
  • http://www.feedforall.com/blog-feed.xml
